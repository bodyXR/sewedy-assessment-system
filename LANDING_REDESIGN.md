### Color System

**Primary Palette:**

- White background (#FFFFFF)
- Deep charcoal/black (#111827)
- Soft gray tones (#F9FAFB, #F3F4F6)
- Muted crimson accents (strategic use only)

**Usage:**

- Background: White and light gray
- Text: Gray-900 for headings, Gray-600 for body
- Accents: Primary red used sparingly
- Borders: Gray-200 for subtle definition

### Typography

**Hierarchy:**

- Hero: 5xl-6xl (48-60px)
- Section Headers: 4xl (36px)
- Subheadings: lg-xl (18-20px)
- Body: sm-base (14-16px)
- Labels: xs (12px)

**Style:**

- Font: Inter (modern grotesque)
- Weight: 400-700 range
- Tracking: Tight for headlines
- Leading: Relaxed for body text

### Spacing

**Consistent Scale:**

- Section padding: py-24 (96px)
- Container max: 7xl (1280px)
- Grid gaps: 6-8 (24-32px)
- Card padding: p-6 to p-8 (24-32px)

## ✨ Motion Characteristics

### Entrance Animations

- **Fade + Translate**: opacity 0→1, y 30→0
- **Duration**: 0.6-0.8s
- **Stagger**: 0.1s between elements

### Hover States

- **Scale**: 1.0 → 1.02-1.03
- **Translate Y**: 0 → -5px
- **Border**: opacity 20% → 40%
- **Glow**: opacity 0 → 100%

### Ambient Animations

- **Orbs**: 8-10s pulse (scale + opacity)
- **Particles**: 10-20s float up
- **Patterns**: 20-60s rotation

### Scroll Effects

- **Parallax**: -100px to +100px
- **Opacity**: Fade in/out based on viewport
- **Scale**: 0.8 → 1.0 on enter

## 🎯 Interactive Elements

1. **Mouse Cursor Glow**: Follows cursor with spring physics
2. **Floating Particles**: 20 particles floating upward
3. **Hover Cards**: Scale + glow on all interactive elements
4. **Scroll Progress**: Sections animate based on scroll position
5. **Navigation**: Glass effect appears on scroll
6. **Smooth Scroll**: Anchor links with smooth behavior

## 📱 Responsive Breakpoints

- **Mobile** (< 768px): Single column, reduced animations
- **Tablet** (768px - 1024px): 2-column grids
- **Desktop** (> 1024px): Full experience with all effects

## 🚀 Performance Notes

- All animations use GPU-accelerated properties (transform, opacity)
- Blur effects are strategically placed
- Scroll listeners are optimized with Framer Motion
- Images should be optimized (WebP/AVIF recommended)
- Consider `prefers-reduced-motion` for accessibility

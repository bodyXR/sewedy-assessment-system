# Logo File Required

Please add your logo image file as `logo.png` in this directory (`public/logo.png`).

## Requirements:

- File name: `logo.png`
- Format: PNG (recommended for transparency)
- Recommended size: 200x200px or larger
- The logo will be displayed at 40px height in the navigation bar

## Current Status:

The navigation component is configured to use `/logo.png`, but the actual image file needs to be added.

## Alternative:

If you prefer a different format (SVG, JPG, etc.), update the image source in:

- `components/landing/navigation.tsx` (line with `<img src="/logo.png"`)
